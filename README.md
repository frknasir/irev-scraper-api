# IREV API Scraper

A script to scrape Nigerian election results from the IREV website.

## Requirements

- Node > 16
- [https://github.com/google/zx](zx)
- Google Sheets API

## Installation

Clone the repo:

```bash
gh clone ...
npm install
chmod +x ./index.mjs
```

Now run it and follow the prompt:

```bash
./index.mjs
```
